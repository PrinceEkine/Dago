    Filter list subscriptions

    Lists are never fetched automatically - enabling one just marks it for use; nothing is downloaded until you press Update. Only https:// URLs are accepted.

      Screenshare relay (TURN)

      Without this, screensharing connects directly peer-to-peer, which reveals both sides' public IP addresses to each other (inherent to how P2P WebRTC works). Configure your own TURN server to relay through it instead - "force relay" refuses direct connections entirely, so a peer only ever learns the TURN server's address.
